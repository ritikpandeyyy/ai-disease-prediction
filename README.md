# AI Disease Prediction & Diet Recommendation (MERN)

## Overview
A full-stack MERN project with AI-powered disease prediction and personalized diet recommendations. Features a modern, responsive UI and modular AI integration.

## Structure
- `/backend` — Node.js/Express API, MongoDB
- `/frontend` — React app (Material-UI)
- `/ai_service` — AI/ML microservice (Python or Node.js)

## Features
- User authentication (JWT)
- Disease prediction (AI-powered)
- Personalized diet recommendations
- Modern, responsive UI
